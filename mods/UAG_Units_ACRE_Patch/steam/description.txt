[h1]What is this?[/h1]

A supplementary patch for the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3263684880]UAG Units mod[/url], adding compatibility with the [url=https://steamcommunity.com/workshop/filedetails/?id=751965892]ACRE mod[/url].

[h1]What does it do?[/h1]

This mod makes the following changes to the UAG Units mod:

[list]
    [*][b]ACRE Radio Items:[/b] UAG Units now come equipped with ACRE radios depending on their role.
[/list]

[h1]How do I use it?[/h1]

This patch requires the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3263684880]UAG Units mod[/url] and the [url=https://steamcommunity.com/workshop/filedetails/?id=751965892]ACRE mod[/url] to function.

Please report any bugs or issues you encounter to the [url=https://github.com/uagpmc/arma3/issues]GitHub issue tracker[/url]. Be sure to include as much detail as possible, including any error messages or screenshots you can provide.
